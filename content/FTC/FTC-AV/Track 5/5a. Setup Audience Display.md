Supplies Needed: None.


1. Open Chrome and navigate to the IP address of the scoring server (this should be the .100 address of the subnet the Streaming Unit computer is part of).
2. If shown a list of events, select the current event. Ask the Equipment Manager or FTA, or use the dates as a point of reference.
	![[FTC/FTC-AV/Track 5/Media/image-1.png]]
3. If you are not prompted to select an event, you will land on the same dashboard as if you were prompted to select an event. Within this dashboard, select "Scoring/Timing Displays" near the bottom of the screen.
   ![[FTC/FTC-AV/Track 5/Media/image-2.png]]
4. You should be directed to a screen with a popup configuration page. Follow these steps carefully to ensure the correct configuration is applied.
	1. The default menu to display is "Display Settings". Within this, set the settings as shown in the screenshot. ![[FTC/FTC-AV/Track 5/Media/image-3.png]]
	2. When you select "Use Overlay", a new sub-menu appears titled "Overlay Settings". Open this sub-menu and click the "Pick Color" button. Set it to transparent.
	   ![[FTC/FTC-AV/Track 5/Media/image 1.png]]
	   Then, click the Pick Color button again to close it. Set the settings as shown in the screenshot.![[FTC/FTC-AV/Track 5/Media/image 2.png]]
	3. Click "Save" to close the menu. Then, click in the URL bar and copy the URL that is present.
Stage 5a is complete. Proceed **immediately** to complete [[FTC/FTC-AV/Track 5/5b. Finish OBS Setup|Stage 5b]].
